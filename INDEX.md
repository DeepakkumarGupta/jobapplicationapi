## ğŸ› Troubleshooting

| Issue | Solution |
|-------|----------|
| MongoDB connection error | Ensure MongoDB is running (`mongod`) |
| Port 3000 in use | Change PORT in `.env` |
| Build errors | Delete `node_modules` and `dist`, run `npm install`, then `npm run build` |
| File upload failing | Ensure file is PDF and under 5MB |
| Docker errors | Run `docker-compose down -v` then try again |
